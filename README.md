# Fetch API: Загрузка данных из публичного API
Проект демонстрирует работу с `fetch`, `async/await` и обработку
состояний загрузки/ошибки.

## 🛠 Технологии
- HTML5, CSS3, Vanilla JavaScript (ES6+)
- Fetch API, GitHub Pages

## 🚀 Запуск
1. Откройте `index.html` в браузере или перейдите по [live-ссылке](https://musa-jusa.github.io/day1-fetch-api/))
2. Данные загружаются автоматически из JSONPlaceholder

## 💼 Для заказчика
- Надёжная обработка сетевых ошибок
- Адаптивная базовая вёрстка
- Чистый, документированный код без зависимостей
